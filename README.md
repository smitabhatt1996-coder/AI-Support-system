# AI Support System

## Project Overview
This project is an AI-based Support System developed to help users get quick solutions to common issues. The system takes user input and provides relevant responses based on predefined logic.

## Features
- Simple and user-friendly interface  
- Accepts user queries  
- Provides instant solutions  
- Handles different types of issues like transcript and coding problems  

## Technologies Used
- Python (Flask)  
- HTML (Frontend)  
- Render (Deployment)  

## AI Usage
In this project, a rule-based approach is used to simulate AI behavior. The system analyzes user input using keyword matching and returns suitable solutions. This helps in automating basic support tasks.
